1. git init, type in terminal

2. 'git remote add origin  <git-url>

    eg git remote add origin https://github.com/Zaleeeeeeee/MERN-Stack

3. to check if linked or not linked: git remote -v

4. git-status to check // compare online-repo and local-repo

5. u - untracked files // new files

6. to add file, git add <file-name>

7. a - added // after git add <file-name>

8. git commit-m "commit"

9. git push origin <branch-name>
    eg. git push origin master

10. git add . to add all files

11. git branch -r

12. git branch *main
13. git checkout -b auth to create a new branch

14. to switch between branches git checkout "name"
        eg: git checkout master
            git checkout auth

15. to merge branches: git merge "branch-name"
    eg: git merge auth //  should be in other branches to change

16. git pull to pull new codes
